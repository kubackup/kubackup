- **简单**：用户可以通过 Web 图形界面轻松管理备份任务；
- **高效**：随着数据的增长，额外的快照只占用实际增量的存储；
- **快速**：使用 go 编程语言编写，充分提升备份效率，备份数据仅受网络或硬盘带宽的限制；
- **安全**：使用 AES-256 对数据进行加密，并使用 Poly1305-AES 对数据进行身份验证，以保证您数据的机密性和完整性；
